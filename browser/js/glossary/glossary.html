Glossary

Action tools

  • Ask, Tell, and Give allow you to talk to a character on the board. Drop a name onto drop person here, and a question, comment, or item onto drop item here.

    Remember, this will only work when you're standing on the same square as the person you want to talk to!

  • Pick up is a special tool that allows you to pick an item up from the board. Drop the name of the item you'd like to pick up onto Drop item here.

Movement tools

  • Move lets you travel around the board. Drop a purple direction block onto drop direction here and enter a number of squares.

    You can't go off the board or through certain objects, like trees and rocks.

Logic tools

  • Drop action here

  • Drop a condition variable onto drop item here, and any tools into drop action here. These nested actions will only run if the condition variable is true.

    For example, a variable that says "the potion is blue" means that the nested actions will only happen if the potion is blue.

  • Drop action here

  • Drop a condition variable onto drop item here, and any tools into drop action here. These nested actions will be repeated until the condition is not true.

    For example, a variable that says "Joe is sad" means that the nested actions will happen while Joe is sad.

  • Drop action here

  • Choose a number and drop any tools into drop action here. These nested actions will repeat for the number of times you chose.

    For example, if you type 3, the nested actions will all happen 3 times.
